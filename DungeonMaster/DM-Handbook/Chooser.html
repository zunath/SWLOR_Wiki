Chooser Menu

The chooser menu in SWLOR functions identically to base Neverwinter Nights. It has several buttons which correspond to the radial menu, but has the added feature of being able to select a specific creature first - whether that be a single left click in the environment, or via the -creatures list as depicted below.
 

Alternatively, you can also click on a character which places a blue segmented selection ring around them:
 

When a character is selected while the chooser menu is open, it will jump to the map and automatically highlight the character's name in the chooser. From there, different options can be chosen to use on the character similarly to the radial menu. Multiple character can also be selected by using a CTRL-click method:

Commands like Limbo will push all NPCs into Limbo. If a combination of PCs and NPCs are selected only NPCs will be pushed into Limbo. 

CTRL-click and drag will use a selection box:
 

Finally, CTRL-clicking the ground will unselect all NPCs
