Dungeon Master Client Basics

The DM Client for SWLOR is a versatile and powerful tool used for running events and overseeing player interaction. Because this tool has a very similar layout to the player client, it allows new quest masters to adjust to the tool very quickly.

New users should note several things:


1. All players and DM Client users are visible in the party screen - this acts as a quick reference player menu which allows DMs/QMs to find online players quickly. Recent connections are generally shown at the bottom of the scrollable list.

2. All player comms are shown in the left-most chat window. If a player uses SWLOR's version of party chat (comms), it will show up here for all DM Client users to see. This allows DMs to keep up with an event party event while they are in a different cell or out of normal volume range. Tells are not displayed to DMs, as they are considered private and only a server administrator can access these for review.

3. Chooser - This button opens the Chooser Menu.

4. Creator - This button opens the Creator Menu.

5. Show Triggers - This button is largely useless on SWLOR due to mechanical overhaul.

6. Appear/Disappear - This button allows a DM/QM to hide or reveal themselves to players for any number of reasons. Depending on whether or not the user is hidden or visible, the button will change accordingly.

7. Pause - This button has been disabled on SWLOR.

8. Difficulty Slider - This tool has been disabled on SWLOR.

Radial Menus

Most interactions with Player Characters and Non-Player Characters on SWLOR is done through Radial Menus, much like on other Neverwinter Nights Servers:

Using the number pad, users can quickly navigate these menus. Buttons on the radial menu correspond to similar placements on the number pad

Limbo - Allows the selected entity to be placed in Limbo. Limbo is an unseen folder where NPCs can be placed when they are unneeded and need to be deleted. They can be recovered by finding the Limbo folder in the Chooser Menu. PCs cannot be sent to Limbo when selecting the Limbo option.

Go To - Allows the user to quickly jump to a PC or NPC when accessed from the radial menu. While this is less useful while the PC in question is in the user's immediate view, the radial menu can also be accessed by right clicking on the PC's portrait in the party sidebar mentioned above.

Heal - Heals the PC/NPC to full health and removes buffs/debuffs. Will also revive a downed PC as long as they don't respawn. Revive does not work on NPCs who have already been defeated.
