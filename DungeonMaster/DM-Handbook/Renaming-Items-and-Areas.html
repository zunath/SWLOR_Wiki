Renaming Items and Areas

Renaming items and areas is a useful storytelling tool that can be used to further immersion within the world of SWLOR and there are two methods to accomplish this: the /rename command and the DMFI Renaming Wand.

Utilizing the /rename command will allow the user to target an object, NPC, or open ground and open an NUI window depicted below:

If the ground is targeted as is depicted in the image above, the area will be renamed. This will be reflected in the Chooser Menu and if a character is in the renamed area, the location tooltip when hovering over their portrait on the party menu.

Additionally, NPC and item descriptions can be changed with the naming wand. A dialogue box will open when an NPC or item is targeted after using the wand, prompting the name or description change:
 

Simply follow the instructions on the dialogue box to complete the name/description change.

For items located in your inventory, the standard /itemdesc command will also change the item's description. Note that there appears to be an approximate 350 character limit to these descriptions.
