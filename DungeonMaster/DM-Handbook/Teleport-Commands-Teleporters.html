Teleport Commands and Teleport Objects

Teleport Objects

Teleport objects are placeable objects that can be placed in the game world by a user to allow quick transition from one teleport object to another. These transitions can be placed within the same area, or in different areas across the server. Upon clicking on this teleport object, the user is instantly teleported via area transition to the corresponding destination.

Teleport objects can be located in the Creator Menu under Placeable Objects > Custom > Special 5.
 

When placed, the teleport object will appear transparent and become highlighted once the mouse cursor hovers over it. From here, the teleport object can be renamed and different destinations and tags can be added with teleport commands.

Teleport Commands

Teleport Commands allow a user to quickly generate area transitions between one teleport object and another. The following teleport commands are available on the server:

/tptag - Allows a user to change a teleport object tag.

/tpdest - Allows a user to change a teleport object destination.

/tpwp - Allows a user to teleport to a transition tag. Note that this only works for pre-placed transition tags included within the base module. User created tags and teleport objects cannot be accessed with this command.

Everything revolves around the initial tag for each teleporter. This essentially tells each teleporter who it is. The destination tag simply tells the teleporter that it's assigned to where the character who clicks on it is going to be sent to. Tags can be set to whatever the user desires, but it's easiest to maintain a singular word that is unique to the specific user, followed by numbers. In this case, tptp[number].
