Taxis and regional travel

Ground travel in busy regions uses taxi terminals and a character-bound Taxi Hailing Device key item. Taxis move you between registered waypoints inside the same region (the server groups terminals by a region id).

Taxi Hailing Device

You must possess the Taxi Hailing Device key item to use player taxi services. On Viscara, the introductory chain Taxi Terminal Repairs awards this item once completed (see the Quick Start Guide). Without the device, terminal dialogs explain that you need to obtain one.

Registering destinations

At a taxi terminal, if you have the device and have not yet registered this stop for the region, choose [REGISTER LOCATION]. That adds the terminal’s destination to your account for that region. You only need to register each stop once per character.

Traveling

After registering, open the same regional network from any terminal in that region. Destinations you have unlocked appear with a credit price; choosing one deducts the fare and moves you to the waypoint. The terminal you are standing at is omitted from the list. Stops you have not registered appear as unavailable.

Dungeon Masters

DMs using taxi dialogs see all destinations in the region without registration or device checks (build/testing support).

Taxis are separate from starship travel, shuttles between planets, and plot-specific transports.
