Release Date 2023-05-08
NWSync Hash 377a91d4a40c0e86c2621ff57e346a60c3ab0d2d

Contributors: Ziya; Xisifer (Iskellia / Daria); Lamarr13; Valo; KingArthur (Grik, Czerka).

Changes

  • Fixed an issue with some beast effects not applying properly.
  • Bug reports are now automatically sent to the bug report forum.
  • Removed several names from the reserved names list: Zell, Luke, and Zayne.
  • A store to purchase retraining tomes and reconditioning tomes with XP has been added. Access it through any training terminal.
  • The Revanite base has been revamped.
  • Nautolans have been added as a playable race.
  • Nautlia has been added as a new language skill.
  • New droid appearances have been added.
  • New stations and ship graphics have been added.
  • Wookiee, Droid, Trandoshan, Zabrak, and Kel Dor characters may now be scaled taller than other races.
  • Call Beast no longer requires stamina.
  • Fixed an issue where the wrong level of Shield Bash was used.
  • New prefab areas have been added.
  • A new high-level dungeon has been added to Hutlar.
  • A revamp to various Force powers and Device abilities has been added.
  • The /tptag and /tpdest DM chat commands have been added.
  • Fixed an issue where the amount of hull or shields restored by the respective module did not display properly in the chat log.
