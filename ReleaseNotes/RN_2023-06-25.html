2023-06-25 Release Notes

Release Date 2023-06-25
NWSync Hash dd2593c3b52f420a31ff61af189927f3dacd9b91

Contributors: Valo; Sytic; Xisifer (Iskellia / Daria); KingArthur (Grik, Czerka); Ziya.

Part 1

  • The server has been updated to support NWN release 8193.35-40.
  • Fixed an issue where Chiro Shuriken and Chiro Rifle items were removed from the module and were therefore uncraftable.
  • Fixed an issue where the Nautolan language was unusable.
  • Twilight Praxeum areas should no longer have weather.
  • Fixed the door in the Sith Crypt to face the correct way.
  • Fixed an issue where a Medical Terminal placeable was usable when it should not have been.
  • Auto-craft has been removed.
  • Fixed an issue where a Beast Stable Terminal could not be placed within properties.
  • Enmity values have been adjusted.
  • Benevolence healing amount has been adjusted.
  • Attacks of opportunity have been disabled.
  • The Mobility perk has been removed and SP has been refunded to anyone who purchased it before this update.
  • Xisifer's armor pack 4 has been added.
  • A merchant selling new training weapons can be found on CZ-220. These weapons are weaker than tier 1 but do not require perks to use.
  • Veles Colony has been redesigned.
  • AOE indicators now display before using abilities.
  • Fixed the description for Force Burst.
  • The player status window can now be clicked through.
  • The transition from the Wildlands to the Lake has been moved to a consistent location.
  • Player city cantina upgrades now reduce the RPXP needed for training terminal items by 10% per level.
  • Player city bonuses now grant bonuses at level 1. Level 5 also grants higher benefits.

Part 2

  • Fixed an issue where the /a command could not be used with beasts.
  • Weapon Enhancements for improving accuracy are now available as recipes under Engineering.
  • Fixed the placement of some bookshelves in CZ-220.
  • Robe 254 has been removed because it was broken.
  • Fixed the Bookshelf, Jedi furniture appearance.
  • Fixed an issue where elemental defense bonuses from food were not showing on the character sheet.
  • The full character rebuild menu now shows scrollbars if the window is too small.
  • The number of resources which spawn in inaccessible locations in Dathomir - Tarnished Jungles North should be reduced.
  • References to the Acid damage type have been changed to Poison for consistency.
  • The /char chat command has been added. It displays your user ID, CD Key, character age, and other pertinent information.
  • Tamed Ssurian beasts are now smaller.
  • Fixed an issue where auras were not removed if the leader died, left the server, or entered space.
  • Fixed the damage for Ion Grenade III.
  • Lightsaber, Saberstaff, and Bulwark blaster deflection has been fixed. Lightsabers and Saberstaffs grant a 5% chance, Bulwark grants 10%, and combining a lightsaber with Bulwark grants 15%. This applies once per round for each ranged attack.
  • Force Spark's sound has been modified to be much less disruptive.
  • Fixed a component for the K'lor'slug Surprise recipe.
  • Searching recipes no longer clears your search text.
  • Fixed a sporadic issue with recipe searching that could cause a crash.
  • Fixed an issue where the Edit Categories permission was not checked when managing item storage categories.
