Changes

  • Fixed an issue where two Racin' Jim's existed in the module, which caused some players to become stuck.
  • Fixed an issue where players would not return to the default respawn point if the area no longer existed.
  • Fixed the name of Metal Wall, Single, Pipes to read Wooden Wall, Planks (Small) to better describe it. Required components were also changed.
  • Fixed an issue where the taxi option for the Veles Apartments took your money but did not teleport you there.
  • Fixed incorrect directions on signs in Veles Colony.
  • Fixed an issue where the Veles Map did not work in the new colony areas.
  • Fixed an issue where the new colony areas did not grant the correct exploration achievements.
  • Fixed several graphical issues on armor pack #4 from the previous update.
  • Fixed an issue where aura effect icons did not display.
  • Fixed an issue where accuracy and damage stats were not displayed on pistols.
  • Fixed an exploit with crafting.
  • Cooking enhancement recipes have moved from Engineering to Agriculture.
