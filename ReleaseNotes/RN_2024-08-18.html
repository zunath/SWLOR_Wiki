Changes

  • Fixed an issue where the Hypermatter Injector did not check for or consume a fuel capsule on use.
  • Fixed an issue with an incorrect Sith appearance.
  • Fixed a potential crash issue in combat calculations.
  • Fixed an issue where Ewokese was not translating properly to all nearby players.
  • Fixed an issue where space loot tables were not set up properly.
  • Fixed an issue where Concussion Grenade damage was not displaying in the combat log.
  • Fixed an issue where the opposite Improved Two Weapon Fighting perk could not be purchased if the other one was refunded.
  • A new recipe category for Special Submission Items has been added to Smithery, Engineering, and Cooking. These items should be used to coordinate upgrades to lightsabers or other DM-granted items with the QM and DM team. Create a ticket before crafting these items.
