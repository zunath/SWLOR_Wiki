Release Date 2022-10-02

Changes

  • Base Stealth Generator recast times have been doubled.
  • The Leadership skill has been increased to a max rank of 50.
  • Additional Leadership perks have been added.
  • Leadership XP can now be obtained by having an aura active or using a leadership ability during combat.
  • Fixed an issue where some shops were not properly updated with the changes to bag stats.
  • Guardian Helmets are no longer marked as stolen. Speak to a DM if you need a replacement after this update.
  • Seats in Racin' Jim's Cantina have been adjusted.
  • Droid ring items have been added to Hutlar stores.
  • Stores selling cooking basics have been added to all crafting areas.
  • Added a quest for acquiring the taxi hailing device key item.
  • Fixed an issue where DMs using the /rename command did not update the database with the target's new name.
  • The Tech Vendor in Veles Colony no longer has a store.
  • Removed unused conversation from Karrav Maubar.
  • Entering locked doors while you have enmity is now prevented until all enmity is lost.
  • Fixed an issue where spaceship mining lasers retrieved more ore than expected under certain situations.
  • Fixed the description of Stasis Field to indicate the correct amount of evasion it grants.
  • Updated starship amplifier modules to clarify they increase attack, not damage.
  • Fixed an issue where the credit reward displayed on guild tasks did not include bonuses granted by SOC and guild relations.
  • Periodic reminders about automated server resets will be sent to all players.
