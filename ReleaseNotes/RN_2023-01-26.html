Release Date 2023-01-26
NWSync Hash Not listed

Contributors: Ziya.

Changes

  • Fixed an issue where the Droid Assembly Terminal required the wrong perk to craft.
  • Fixed an oversight where droid instruction disks were not craftable.
  • Fixed an issue where the Jungle Bugs on Dathomir did not deal the correct damage.
  • Fixed an issue where the perk refund confirmation still showed 12 hours instead of 1 hour.
  • Fixed an issue where Drill Calamary bait could not be loaded onto fishing rods.
  • Most Force perks have been reduced in SP cost. Spent SP has been refunded so the perks can be repurchased at the new price.
  • Fixed an issue where apartments you have access to could occasionally not display in the menu.
  • Grenades and bombs now have separate cooldown timers.
  • Broadcast terminals can be found in NPC cities. These broadcast messages to logged-in players and post to Discord for a nominal fee.
  • A /bc command has been added for DMs. This broadcasts messages in-game and posts them to Discord.
  • Fixed an issue where property fees were only sporadically billed on boot up.
  • The /time command now displays in-game time in addition to server time.
