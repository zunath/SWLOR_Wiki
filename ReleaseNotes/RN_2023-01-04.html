Release Date 2023-01-04
NWSync Hash Not listed

Contributors: Valo.

Changes

  • Fixed an issue where non-DM players could get as far as selecting a target when using a DM command.
  • Added a test mode which grants regular players access to most DM commands on the test server.
  • Bug reports sent from the test server are now suffixed as test server reports.
  • Fixed an issue with Dathomir boss spawn times occasionally being incorrect.
  • Droid armor crafting has been added to the Engineering recipe list.
  • Fixed an issue with some recipes.
  • Fixed an issue where Alacrity and Clarity triggered too often.
  • Fixed an issue where the AI did not stick to a target for 3 rounds as expected.
  • The +Duration property on food now increases duration by 5 minutes per bonus, up from 1 minute.
  • Fixed an issue where Flamethrower's DC increased based on how many enemies were in the AOE.
  • A welcome message with a link to the Discord now shows on login.
