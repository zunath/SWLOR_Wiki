Changes

  • New creature appearances have been added. These are currently available only for DM use and will be incorporated at a later date.
  • The /changeitemdescription command has been renamed to /description and /desc. A pop-up window now displays for entering the description. DMs can also use this to change the description of any object in the game.
  • The /ordercompanion command has been added. Alternates are /order and /oc. This lets players tell a companion, droid, or beast to attack a target, stop what they are doing, or open a door.
  • Corrected an issue with module attachment.
  • Fixed an issue where character equipment was taken into account for starship combat.
  • Texture changes from the previous release have been reverted to help address crashing issues.
  • August 25 event content has been added.
  • Fixed an issue where Ardanium asteroids were not spawning as expected.
  • A hole in the mountain range in Dantooine Canyon has been fixed.
  • A floating frying pan rack has been fixed.
