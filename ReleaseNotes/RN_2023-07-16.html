Release Date 2023-07-16
NWSync Hash Not listed

Changes

  • Improved database caching performance.
  • Improved perk level check performance, including equipping and unequipping items.
  • Perks are now automatically refunded when a skill decays. This only refunds specific levels, not the entire perk line. Use retraining tomes to refund an entire perk.
  • Any perks that no longer satisfy requirements will be removed on login.
  • Fixed an issue where teleporter objects could be used while on a creature's enmity table.
  • Dash can now be refunded while active. It will be toggled off when the refund completes.
  • Stabling can now be refunded even if you have more than one beast in the stables. You will be unable to switch beasts until you either release another beast or repurchase the perk.
  • Equipment now automatically unequips if the associated armor perk is refunded.
  • Fixed an equipment bug.
  • Reverted spawn system changes from the previous update.
  • Added the /follow chat command.
