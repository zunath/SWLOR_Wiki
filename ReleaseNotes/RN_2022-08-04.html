2022-08-04 Release Notes

Release Date 2022-08-04
NWSync Hash Not listed

Contributors: Ranger X; Wildkins (Maya); Xisifer (Iskellia / Daria); Solitude; John Republic; Valo; Marrowow [Jac/Neena/Efri].

Changes

  • Applied walkmesh fixes to the Korriban Sith Crypt area and adjusted ghost sound sets.
  • Fixed an issue which caused guildmaster quests to show as accepted rather than available.
  • Various Mandalorian changes were made to loot, items, and related content. More detail: SWLOR_NWN pull request 1467.
  • Treasure Hunter now increases the chance to pull from rare loot tables. Previously this was on a per-item basis; now it affects entire loot tables.
  • Fixed an issue where HP would not refresh after being healed.
  • Fixed an issue where players could remain immobilized after logging back in from the middle of a Holocom call.
  • Fixed an issue where DM speeds were slower than expected.
  • Holocom has been disabled while in space, and players cannot call someone who is in space mode.
  • Fixed an issue where Dash and Burst of Speed caused inconsistent speeds.
  • Tranquilizer Shot III's description now states that it targets up to three creatures in a cone.
  • Tranquilizer Shot III can no longer target the activator.
  • Rapid Shot's description now states that it only works with pistols and rifles.
  • Rapid Reload's description now states that it allows rifles to gain APR but does not grant APR itself.
  • Combat log text has been clarified for PC and NPC to-hit rolls.
  • Fixed an issue where the dates and times on the manage citizenship window were unreadable.
  • A new female armor set has been added.
  • Various fixes were made to female models.
  • XP debt now displays on the skills menu.
  • A message is now sent when XP debt is added to your total.
  • XP debt now caps at 9,999,999.
  • Fixed an issue where low-powered module icons were not cleared when removed from ships.
  • Fixed an issue where evasion-lowering abilities, such as Saber Strike, Force Spark, and Cross Cut, did not lower enemy evasion.
  • Increased the range of Circle Slash and Spinning Whirl from 1.67 meters to 5 meters.
  • Fixed the wrong last name being sent in combat logs.
  • Circle Slash and Spinning Whirl now properly hit up to three targets in the sphere instead of the same enemy multiple times or no enemies at all.
  • Increased the AC reduction duration on poison status ticks from 1 second to 6 seconds.
  • The refinery process has been revamped. Raw materials can now be mass refined instead of processed one at a time.
  • A new buildable area has been added to Hutlar.
  • New achievements have been added.
  • Tatooine travel times have been shortened.
  • Crafting devices, guildmasters, and a bank have been added to Tatooine.
  • Fixed an issue where attacks per round were not reapplied properly on login.
  • DMs are now hidden from the player list.
  • Fixed the descriptions for Battle Insight and Force Stun.
  • Bonuses to Accuracy from status effects have been increased.
  • SOC now reduces XP debt by 3% per point above 10. The overall XP debt reduction cap is 80%.
  • Added crafting gear recipes.
  • Added new DM NPCs.
  • Adjusted female Rodian heads.
  • Added two new female Chiss, Echani, Human, and Mirialan heads.
  • Training Saberstaffs can now only be crafted by Force Sensitive characters who have learned the recipes.
  • New armor appearances have been added.
