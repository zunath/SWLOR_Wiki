Release Date 2025-08-04
NWSync Hash e9c59001c7d17c5410d9a254ddda5689a96f6f89

Changes

  • Fixed an issue where Charge would overwrite or disable Dash when it wore off.
  • Fixed an issue where the NWN Bleed and Disease effects were not removed by treatment kits.
  • The rest effect is now properly removed if you attack something.
  • Fixed an error if certain UI windows were closed in specific ways.
  • Holocom notifications stop sending if the sender logs out of the server.
  • Pending Holocom calls now cancel if the /endcall command is used.
  • Fixed an issue where items placed inside a refinery were lost if you walked away without removing them first.
  • Fixed an issue where players remained busy after trying to use an exhausted fishing point.
  • Fixed calculation issues on the character sheet while Flurry Style is active.
  • Harvester XP is now based on the number of items gathered.
  • Fixed an issue where associates did not use Resuscitate.
  • Fixed an issue with the evasion bonus applied for Soldier's Speed.
  • Fixed a few missing icons.
  • Added /resetwindows, which resets all SWLOR windows to their default position and size.
  • A level filter has been added to the Recipes window.
  • An option to only show craftable recipes has been added to the Recipes window.
  • Fixed an issue where the Tatooine Cantina Bartender did not show a dialog to purchase drinks.
  • Stealth now properly breaks when offensive actions are used.
  • The market buy menu now has an option to sort pricing from low-to-high or high-to-low.
  • Resources now despawn after a certain amount of inactivity.
  • Players can no longer distribute more RP XP than a skill needs. The XP limit is also displayed on the window.
  • The Next and Previous buttons in the Market Buy menu now work again.
  • Loadscreen hints have been updated.
