Contributors: Archie - HK32, Laius Keth; Scorchys; Tinfy (Alora Vaas); Zickery; Ziya; Valo; MachineMind.

Changes

  • Applied a fix to the Shield Master perk.
  • Server restart reminders now go directly to your combat chat log instead of floating above your PC's head.
  • An option to enable or disable server reset warnings has been added to the settings menu.
  • Text on medical registration devices has been updated to be more lore-appropriate.
  • New droid parts have been added.
  • The max range of Force Lightning I has been increased to match tiers 2 and 3.
  • Fixed an issue where an incorrect item was defined for the Surprise Sandwich Agriculture Guild quest.
  • Fixed an issue where the incorrect component for K'lor'slug Sandwich was defined.
  • Fixed the description of Strong Style to match the design bible.
  • Fixed the description for tier 2 of Inner Strength.
  • A taxi terminal for Racin' Jim's Cantina has been added.
  • NPC shuttles have been removed from the spawn list of space encounters.
  • An incorrect description of an NPC in Qion Box Canyon has been removed.
  • Tofu, Tomato, Turnip, and Viscaran Peas are now stackable.
  • Starship Shield and Hull Repairers now target yourself if you do not have a target specified.
  • Fridge, Stainless and Weapon Rack now grant +1 item storage as expected.
  • Dathomir has been added.
  • 37 new Smithery recipes have been added.
  • 15 new Engineering recipes have been added.
  • 100+ new items have been added.
  • The number of players required to start and maintain a city has been reduced for all tiers: Level 1 = 5, Level 2 = 10, Level 3 = 15, Level 4 = 20, and Level 5 = 25.
  • 50+ new cooking recipes have been added.
  • Fishing has been added to the Agriculture skill line.
  • A new dungeon has been added to Korriban.
  • New quests have been added to Mon Cala.
  • New Agriculture Guild quests have been added.
  • Fishing points have been added to all planets.
  • New Fabrication recipes have been added.
  • XP and credit rewards have been increased for one-shot quests on Viscara.
  • Player buildings, cities, apartments, and ships are now prefixed with {PC} to help DMs find them.
  • Creature AI improvements have been made.
  • Torso armor pieces 154 and 155 have been adjusted for Human Male and Human Female.
  • A new female portrait has been added.
  • The voice and portrait of the Fabrication Guildmaster have been fixed.
  • Fixed an issue where Flurry Style stat changes were referenced on all weapon types instead of just Staves.
  • Shielding description has been updated to reflect that it boosts physical defense.
  • Benevolence activation and recast times have been adjusted.
  • Fixed an issue where HP, STM, FP, Shield, Hull, and Capacitor bars could go beyond the bounds of the bar.
  • Fixed an issue where Deflector Shield, Burst of Speed, Force Spark, and Disturbance could be stacked.
  • A new Leadership perk, Shout Range, has been added. This increases the range of your auras.
  • Fixed the conversation for Farah Oersted's quest line.
  • DM-specified area notes now only display if one has been set.
  • Guylan Verruchi's quest in Hutlar Outpost has been fixed.
