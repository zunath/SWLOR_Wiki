2022-08-13 Release Notes

Back to Release Notes

Release Date 2022-08-13
NWSync Hash Not listed

Contributors: Dubh | Sheridan; Xisifer (Iskellia / Daria); MachineMind; Wildkins (Maya); John Republic; Valo; Oda Nobunaga; Noosehallo.

Changes

  • Added a new portrait to Human male.
  • New armor parts were added to races based on Elf, Dwarf, and Half-Orc.
  • Fixed the recipe for Forza Cloaks.
  • Added a missing key to the PC Mandalorian Base.
  • Enemies must now have line of sight on the user to be affected by Provoke.
  • Tweaked aggro AI to require line of sight. This should help areas like the Coxxion Base where enemies were aggroing through walls.
  • Fixed an issue where the Throwing Mastery perks were not applying to throwing weapons.
  • Changed the icon of the Amphi-Hydrus Brain Stem item. Existing items will not be updated; only new ones are affected.
  • Hutlar is now 3 degrees colder.
  • Explosives now require line of sight to use on a target.
  • Fixed an issue with DMFI tools not working for DMs.
  • Fixed female robes and loincloths not animating properly.
  • Fixed an issue where the On Hit Cast Spell item property would not always apply to rifle and blaster ammo.
  • Fixed an issue where mining asteroids granted less XP than expected.
  • Fixed an issue where subdual mode considered the dying target rather than the attacker.
  • DMs can now see more detailed information about player characters.
  • DMs can now store notes about player characters.
  • Fixed some transitions in Tatooine that led to incorrect locations.
  • Crafted items now sell for more than 1 credit.
  • Added new DM event areas.
  • Hunter's Guildmaster has been added to Tatooine.
  • Guildmasters have been added to Hutlar Outpost.
  • Replaced Cannons with Shurikens in the Ranged skill description.
  • Rebuilds may now be performed at any Training Terminal. These cost one Rebuild Token. All characters received one rebuild token as part of this update. Additional tokens may be granted as DM rewards.
  • Droids have been added as a playable race. Voices are limited for this release.
  • Droids may only equip items specifically made for droids. Organic characters cannot equip droid equipment.
  • Droid Equipment perks have been added to Engineering.
  • Racial bonuses and penalties have been removed. Characters now receive one racial point to distribute to any one stat. Once distributed, this racial bonus cannot be removed, even with a rebuild.
  • A /notes command has been added for DMs. This can be used to assign public and private notes to an area.
  • If an area has a DM note assigned, a message will be sent to players upon entry.
  • Female Zabrak head positions have been slightly adjusted to fit on the neck correctly.
  • Adjusted the bust size on several female armor torso options.
  • Pricing on various items has been adjusted.
  • Fixed an issue where a lower tier ability used a higher tier ability's effectiveness.
  • Added Throw Rock to the Force skill.
  • A business owner on Viscara has relocated his shop.
  • Fixed two broken Smithery Guild quests.
  • Added the Medical Exam Bed recipe to Fabrication.
  • Added T2 vegetable spawns to Korriban.
  • Added T2 herb and vegetable spawns to Mon Cala and Viscara.
  • Added T5 herb and vegetable spawns to Tatooine.
  • Jasioclase spawn percentage has been increased slightly.
  • The neo-crusader armor texture has been adjusted.
  • Fixed an issue where accuracy, attack, defense, and evasion during combat did not apply effect bonuses such as Combat Enhancement.
  • Made a number of items stackable.
  • Gimpassa has been nerfed.
  • Mando Hunter has been nerfed.
  • Fixed refinery XP gains.
