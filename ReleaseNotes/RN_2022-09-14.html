Release Date 2022-09-14
NWSync Hash Not listed

Contributors: MachineMind; Wildkins (Maya); Lance (Elyd, Sarkell); acidchalk; Xisifer (Iskellia / Daria); Valo.

Changes

  • Added the /cm command for DM use. This lets DMs save and load creatures.
  • Added the /emotegui command, which opens a window for using emotes.
  • Tranquilizer now inflicts Sleep instead of Daze.
  • New lightsaber models have been added.
  • Fixed an issue where new Echani characters could not pick a class during character creation.
  • New droid part appearances have been added.
  • Fixed an issue where the ability name was not displaying on the AP upgrade pop-up window.
  • Mon Cala and Korriban now sell tier 3 droid equipment.
  • A Food category has been added to the player market menu. Existing items will need to be relisted to appear in this category.
  • Fixed an issue where Other ships inconsistently loaded in the ship management window.
  • Xisifer's armor pack #3 has been added.
  • Space spawns have been added and tweaked.
  • Loot table adjustments have been made.
  • Item pricing adjustments have been made.
  • Coriander, Cumin, Cayanne, and Tumeric have had their bonuses corrected.
  • New instances of Qion Tiger Fangs can now be sold to vendors.
  • Market listings will be removed on the first restart after two weeks of being listed. Existing listings will expire two weeks from this update.
  • Fixed the appearance of Byysk Longswords.
  • Removed an unused transition in the Jawa Machine area of Tatooine.
  • The weight reduction bonus on extra large bags has been corrected. Contact a DM for a replacement if needed.
  • Players can now click effects to send information about them to the combat log.
  • The /alwayswalk chat command has been added. The alternate command is /walk.
  • Various effect icons have been adjusted.
  • Base game references to gold have been replaced with credits.
  • Typing into the chat bar now displays a chatting indicator above player heads.
  • Throw Rock IV and V can no longer target yourself or friendlies.
  • Fixed an issue where enemy corpses continued to be factored into aggro.
  • Fixed an issue where chat commands were considered Holonet postings and subjected users to the 5 minute delay.
  • Wildkins' balance changes have been applied. More detail: SWLOR_NWN pull request 1487.
  • A rebuild token has been granted to all characters.
