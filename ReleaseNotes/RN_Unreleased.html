Next Release

  • Guild quests now require you to be the producer of the item turned in. For example, you must craft or fish it up. Hunter's Guild quests are unaffected by this change.
  • Dantari Shaman enemies now count towards quest progression as expected.
  • Full rebuilds now let you reassign your racial stat bonus.
  • Fixed an issue where dicebags could not be used by DM-possessed creatures.
  • A new window for guild tasks has been added.
  • Katars now use the unarmed animations.
  • /bug should be properly posting to Discord once again.
  • DM and player broadcasts should be operational again.
  • Details about player cities will be posted to Discord in the property-broadcasts channel. These include upcoming elections, election results, new cities, and similar notices.
  • An issue where an agriculture task could not be completed has been fixed.
  • Incubation eggs now have an increased sale price to NPC vendors.
  • Fixes to a few assets.
  • Notifications will now be sent to server-notifications when the server starts booting, is fully booted, restarts manually by a DM, or automatically restarts.
  • There is no longer a rate limit on bug reports on the test server.
