This section is intended for developers of customization parts, such as player heads, helmets, etc.

 

This list should be kept up to date with each new patch adding customization pieces.

 

The lists should be used as a reference guide when choosing which slots to “replace” with new assets. This is done in order to avoid going over the hardcoded 255-part limit that NWN:EE has per part, per gender, per phenotype. For example, there are 255 Male Human heads, 255 Female Elf heads, 255 Male Half-Orc torsos, etc.

The player races in SWLOR use the following phenotypes:

  • Human: Human, Echani, Mirialan, Chiss, Mon Calamari, Trandoshan, Twi'lek, Togruta
  • Elf: Bothan, Wookiee, Rodian
  • Dwarf: Ugnaught
  • Half-Orc: Cathar
